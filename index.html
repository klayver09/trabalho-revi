1.Cascata

Cascata é o princípio do CSS que define qual estilo será aplicado quando houver conflito entre regras.

  • 1.Importância (!important vence tudo)
  • 2.Especificidade
  • 3.Ordem de aparição (o último sobrescreve se tudo mais for igual)

    • Exemplo

    2. Especificidade

  • Inline → 1000 pontos (ex: style="color: red")
  • ID → 100 pontos (ex: #meuId)
  • Classe, atributo, pseudo-classe → 10 pontos (ex: .classe,
  • Elemento, pseudo-elemento → 1 ponto (ex: p, h1, ::before)

    3. Combinadores

  • Descendente (espaço) → seleciona todos dentro.
  • Filho direto ( > )
  • Irmão adjacente ( + )
  • Irmão geral ( ~ )

    • 4. Box Model

    Todo elemento é uma caixa no CSS, formada por:

  • Content → conteúdo (texto, imagem).
  • Padding → espaço interno (entre conteúdo e borda).
  • Border → borda da caixa.
  • Margin → espaço externo (entre caixas).

    5. Display

    Define como a caixa se comporta na página.

  • block → ocupa toda a linha (div, p, h1).
  • inline → ocupa só o conteúdo, não quebra linha (span, a).
  • inline-block → mistura dos dois (respeita largura/altura mas fica em linha).
  • flex → cria layouts flexíveis.
  • grid → cria layouts em grade.

    • Exemplo

  • Cascata → ordem de aplicação.
  • Especificidade → seletor mais forte vence.
  • Combinadores → relacionam elementos.
  • Box Model → estrutura da caixa (content, padding, border, margin).
  • Display → como a caixa aparece (block, inline, flex, grid).
